# Changelog

All notable changes to Solar Helper are documented here.

## [1.3.0] - 2026-02-24

### Added
- Freeze input setting — stops player movement while sending messages
- OpenRouter AI fallback for unscrambling when primary AI is unavailable
- Configurable toggles for typos and math fuzzing (typos off by default)
- Delay to sellall so input freeze applies correctly

### Changed
- Stricter AI prompt for unscrambling — system message enforces single-word responses only
- Lowered default delays across the board

### Fixed
- Update checker now only notifies when a release is actually newer

## [1.2.0]

### Added
- Mod Menu integration with YACL config screen
- Human-like typing simulation
- Math answer fuzzing

## [1.1.0]

- Initial release — auto-welcomer, sellall, math solver, and unscrambler for Solar Skies
